# Optimize

本目录保存质量扫描、维护性审计和治理建议。

## 使用规则

- 用于记录当前问题、风险、建议和下一阶段治理方向。
- 如果建议需要执行，升级为 `docs/plans/NNNN-*.md`。
- 已转为计划的 optimize 文档应链接对应 plan。
- 不把 optimize 文档当作长期架构约束；长期规则进入 `docs/ADR/` 或 `docs/architecture/`。
