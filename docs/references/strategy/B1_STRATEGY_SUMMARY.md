# B1 战法策略总结

> 本文基于当前仓库中的 `SKILL.md`、`knowledge/`、`references/research/` 与 `modules/indicators.py` / `modules/strategies.py` 整理。用途是把 B1 战法从语料规则落到可计算指标；不构成任何投资建议。

## 1. 策略定位

B1 是回调波首次买点，本质是在上涨结构或潜在建仓结构中等一次低风险试错点。

- 盈亏比：高，错了可以用较近止损退出。
- 确定性：低，需要后续 B2/B3 确认。
- 操作方式：次日慢慢买，分批建仓；常见口径是开盘一笔、水下补一笔，或 5k/5k 分批。
- 核心纪律：B1 后 3-4 个交易日不涨就拍掉；牛逼的 B1 后面应有 B2 确认。

## 2. 判定口径

当前仓库存在两套 B1 口径，应区分使用：

| 口径 | 用途 | 触发条件 |
|------|------|----------|
| 严格交易触发 | 真正执行 B1 买点 | `J <= -10`，缩量回调，不有效跌破 60 日线 / BBI / 黄线等关键支撑，避开连续绿砖下跌 |
| 量化初筛 | 扫描候选池 | `J < 13`、振幅较小、涨幅落在区间、成交量缩小，4 项满足 3 项即 `is_b1=True` |
| N 型结构辅助 | 判断回调质量 | 缩量、J<20 或 RSI<20、回调幅度约为第一波的 1/2-1/3、不破前低 |

实际使用优先级：先用严格交易触发定买点，再用量化初筛扩大观察池。

口径校验说明：交易文档与知识库采用 `J <= -10`；`modules/strategies.py` 的实现是 `if j >= -10: return None`，所以代码实际触发为 `J < -10`。两者只在 `J` 恰好等于 `-10` 的边界点不同，本文执行口径统一按 `J <= -10` 描述，代码实现处单独注明。

注意：`J < 13` 是正数 13，用于候选初筛或白黄线选股后的低位信号；`J < -13` 是长安战法第一天的更极端条件，不能把两者混为一条 B1 规则。

## 3. 指标公式与取值

下表覆盖 B1 判定、筛选、过滤和执行中用到的主要指标。符号约定：`C`=收盘价，`O`=开盘价，`H`=最高价，`L`=最低价，`V`=成交量，`PC`=昨收价。

| 指标 | 计算公式 | B1 取值 / 阈值 | 说明 |
|------|----------|----------------|------|
| `MA_N` | `MA_N = SUM(C, N) / N` | 作为 BBI、黄线、周线过滤的基础 | 简单移动平均。 |
| `EMA_N` | `EMA_t = C_t * K + EMA_{t-1} * (1-K)`，`K=2/(N+1)` | 白线使用 `N=10` 连续平滑两次 | 指数移动平均，代码初值取首个价格。 |
| `RSV` | `RSV = (C - LLV(L,9)) / (HHV(H,9) - LLV(L,9)) * 100`；分母为 0 时取 50 | KDJ 参数 `9,3,3` | KDJ 的基础值。 |
| `K` | `K_t = 2/3 * K_{t-1} + 1/3 * RSV_t` | 初值 50 | KDJ 慢化值。 |
| `D` | `D_t = 2/3 * D_{t-1} + 1/3 * K_t` | 初值 50 | KDJ 再慢化值。 |
| `J` | `J = 3K - 2D` | 严格 B1：`J <= -10`；代码触发边界：`J < -10`；量化初筛：`J < 13`；长安战法首日：`J < -13`；N 型辅助：`J < 20` | B1 最核心指标。负值越深，越接近超卖低点，但必须配合结构和缩量。 |
| 涨跌幅 `pct_chg` | `pct_chg = (C - PC) / PC * 100%` | 完美图形统计：`-2% ~ +1.8%` | B1 当日不追涨，偏小阴小阳或十字星。 |
| 振幅 `amplitude` | `amplitude = (H - L) / PC * 100%` | 量化初筛：`< 4%`；完美图形上限：`<= 7%` | 越窄越好，表示分歧收敛。 |
| 缩量 `b1_volume_shrink` | `V_t < V_{t-1}` | `True` | 当前指标引擎的 B1 评分项。 |
| 极致缩量 `is_suoliang` | `V_t <= 0.5 * V_{t-1}` | 最佳 B1 加分项 | 策略引擎把缩量回调作为高置信条件。 |
| 倍量 `is_beidou` | `V_t >= 2 * V_{t-1}` | B1 后若出现，可作为 B2 确认观察 | B1 本身不要求倍量，B2 才重点看放量。 |
| BBI | `BBI = (MA3 + MA6 + MA12 + MA24) / 4` | 严格执行：不有效跌破 BBI；代码候选：记录价格是否在 BBI 下方或附近 | 旧版少妇战法的多空线和支撑过滤。若跌破后只属于低位候选，不能单独视为严格 B1。 |
| 60 日线 | `MA60 = SUM(C,60) / 60` | 不破 60 日线，或在 60 日线附近 | B1 常用防守参考。 |
| 白线 | `WHITE = EMA(EMA(C,10),10)` | 右侧票要求白线在黄线之上；回踩白线 + B1 是强势买点 | 短期趋势线，破白线常是第一道警戒。 |
| 黄线 / 大哥线 | `YELLOW = (MA14 + MA28 + MA57 + MA114) / 4` | 白线在黄线上才重点关注 B1；回踩黄线 + B1 交易价值最大 | 主力成本线 / 多空分水岭。 |
| 黄金碗 `is_in_bowl` | `WHITE > YELLOW AND YELLOW <= C <= WHITE` | `True` 时容错率较高 | 白线和黄线之间的区域；碗大时甚至不必强等极致 B1。 |
| 白黄金叉 | 昨日 `WHITE <= YELLOW` 且今日 `WHITE > YELLOW` | 近 20 日白线上穿黄线后，再等 `J < 13` | 趋势由空转多的过滤条件。 |
| 白黄死叉 | 昨日 `WHITE >= YELLOW` 且今日 `WHITE < YELLOW` | 出现后不做 B1，已有仓位也要高度警惕 | 趋势走坏信号。 |
| 周线多头补丁 | `MA55 > MA144 AND MA144 > MA233`，且 55/144 周线趋势向上 | 必须优先通过 | 用来过滤下跌趋势票。 |
| 累计换手率 | `turnover_sum = SUM(turnover_i)`，重点看前三根中大阳线累计换手 | `< 38%` | 超过通常视为筹码发散、主力可能已跑。 |
| 自由换手率 | `free_turnover_sum = SUM(free_turnover_i)` | `< 62%` | L2 数据口径。 |
| 建仓波涨幅 `b1_rally_pct` | `b1_rally_pct = (C_today - C_low_start) / C_low_start * 100%` | 代码通过区间：`25% ~ 40%`；语料口径约 30% | 当前实现用最近 30 天最低点附近作为估算起点。 |
| 买入区间 | `(最高买入价 - 最低买入价) / 最低买入价 * 100%` | `<= 5%` | 震荡市补丁，防止成本拉太散。 |
| 回调幅度 | `pullback = (第一波高点 - 回调低点) / (第一波高点 - 第一波低点)` | `1/3 ~ 1/2` | N 型结构辅助条件；过深容易破坏结构。 |
| 前低防守 | `L_today >= previous_swing_low` | 不破前低 | 跌破前低后需按 SB1 / 超级 B1 另行判断。 |
| 四绿砖过滤 | 最近 4 根 K 中阴线数量 | 连续下跌状态不抄底；需翻红且不创新低 | 策略引擎用 `yin_count_4` 记录近 4 根阴线数。 |
| B1 评分 `b1_score` | `score = [J<13] + [amplitude<4] + [-2<=pct_chg<=1.8] + [V_t<V_{t-1}]` | `score >= 3` 则 `is_b1=True` | 当前 `detect_b1_today` 的量化初筛规则。 |

## 4. 标准 B1 流程

1. 先看结构：只做主线票或主题票；右侧交易优先要求白线在黄线之上，或者至少没有明显破坏 BBI / 60 日线。
2. 再看位置：回踩白线、落入黄金碗、回踩黄线三类位置优先，回踩黄线 + B1 交易价值最高。
3. 再看 J 值：严格买点看 `J <= -10`；候选观察可以放宽到 `J < 13`，但不能把初筛当买点。
4. 再看量价：缩量回调、小阴小阳、振幅小、涨跌幅不大，越接近地量和十字星越好。
5. 最后看补丁：周线 55/144/233 多头、前三根中大阳线累计换手 <38%、自由换手 <62%、买入区间不超过 5%。

## 5. 入场、止损与退出

| 项目 | 规则 |
|------|------|
| 入场 | 次日分批慢买；不要集合竞价梭哈，可开盘 1/3、9:40-9:45 第二手、10 点第三手。 |
| 止损 | 买入 K 线最低价、N 型结构前低、横盘平台下沿三选一；常见止损幅度 3%-5%。 |
| 时间止损 | B1 后 3-4 个交易日不涨就拍掉或减仓。 |
| 趋势止损 | 破白线是第一警戒，破黄线是更强离场信号；白线死叉黄线后不再按普通 B1 博弈。 |
| 确认 | B1 后 3 个交易日内若出现 B2 放量中长阳，买点被确认；否则降低预期。 |

## 6. 变体与例外

### 超级 B1

形态：N 型上涨结构中，缩量回调到极致后，突然出现一根放量大阴线击穿止损位，随后 1-3 天缩量企稳，`J` 仍为负值，并出现十字星或小阳反转。

主要公式与取值：

- 放量大阴线：`pct_chg <= -3%`，`V_t / V_{t-1} >= 1.5`，且 `C < O`。
- 大阴线后缩量：后续 1-3 天成交量均 `<= 大阴线成交量 * 0.7`。
- J 值：`J <= -5`，越负越好。
- 反转 K：实体幅度 `abs(C-O)/PC <= 2%`，或小阳线。
- 止损：放量下跌那根 K 的最低价；只赌一次。

### SB1 假摔

形态：B1 后跌破前低，再快速收回。

主要公式与取值：

- 假摔：`L_fake_drop < previous_low`。
- 收回：次日或后续收盘 `C > previous_low` 且涨幅 `> 2%`。
- 放量：收回日成交量 `> 假摔日成交量 * 1.2`。

## 7. 不做清单

- 白线在黄线下方，且没有左侧估值逻辑支撑的票，不按右侧 B1 做。
- 连续上涨后的第一个 B1 慎做，第二、第三个 B1 更稳。
