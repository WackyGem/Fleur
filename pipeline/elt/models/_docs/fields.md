{% docs field_security_code %}
A 股证券标准连接代码，格式为 `<6位证券代码>.<交易所代码>`，例如 `601088.SH`。当前交易所代码允许 `SH`、`SZ`、`BJ`。
{% enddocs %}

{% docs field_security_local_code %}
证券在交易所内使用的 6 位本地代码，不包含交易所代码。
{% enddocs %}

{% docs field_exchange_code %}
A 股交易所代码，当前支持 `SH`、`SZ`、`BJ`。
{% enddocs %}

{% docs field_trade_date %}
A 股市场交易日期。
{% enddocs %}
